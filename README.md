## 🌟 Overview
Got a new Mac? Be ready to code in minutes. This comprehensive bootstrap script automatically installs Homebrew, sets up Git configurations, downloads essential CLI tools, and installs popular IDEs and browsers.

## ✨ Key Features
- 🍺 **Homebrew Automation**: Installs the package manager silently.
- 🛠️ **Dev Tools Bundle**: Installs Node, Python, Go, Docker, and more.
- 🎨 **App Installations**: Fetches VSCode, iTerm2, and Chrome via Casks.

## 🚀 Getting Started

### Prerequisites
Make sure you have `Bash` installed on your local machine.

### Installation
```bash
git clone https://github.com/HafezMosleh/mac-dev-setup.git
cd mac-dev-setup
chmod +x main.sh
```

### Usage
```bash
./main.sh
```
