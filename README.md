# mac-dev-setup

Automated macOS setup script for developers (Homebrew, Dotfiles, VSCode configs).

## Features
- 🚀 Fast and optimized
- 🔒 Secure by default
- 🛠 Easy to configure

## Installation
```bash
git clone https://github.com/HafezMosleh/mac-dev-setup.git
cd mac-dev-setup
```
